# 파이프라인 코드

실험은 Google Colab Pro+ 환경에서 진행되었고, 하드웨어 가속기는 NVIDIA A100 GPU가 사용되었다. 
소프트웨어 환경은 Python 3.12를 기반으로 하며, 딥러닝 프레임워크는 PyTorch 2.8.0 (CUDA 12.6)이다. 

**pipeline.ipynb**

- ```이상치 점수(D-score)```: 코드가 형태적으로 학습 데이터의 일반적 분포에서 얼머나 벗어났는지를 나타내는 점수
 1. CodeBERT 모델을 활용해 임베딩 추출
 2. 차원 축소(PCA)를 통해 노이즈 제거 및 연산 효율화
 3. Isolation Forest를 사용해 이상치 탐지

- ```일관성 점수(RI-score)```: 동일한 입력에 대해 모델이 생성한 코드들의 평균 편집 거리를 측정한 점수
 1. 자연어 데이터를 모델에 입력하고 5번 코드를 생성
 2. 생성된 코드 간의 Levenshtein 거리 평균 계산

- ```CVSS 점수(C-score)```: 코드의 문맥에 따라 직접 산정한 CVSS 점수
 1. 코드 문맥에 따라 CWE 번호를 부여
 2. 해당 CWE와 코드 문맥을 고려하여 8가지 항목(공격 경로, 공격 복잡도, 필요 권한, 사용자 관여, 범위, 기밀성, 무결성, 가용성)에 따라 점수 산정

- ```통합 점수(R-score)```: 이상치 점수, 일관성 점수, CVSS 점수를 모두 합한 점수

- 통합 점수가 높을수록 고위험 데이터로 분류되며, 상위 k%의 데이터를 제거해 전처리한 다음 모델을 학습시킨다.
